A full-stack web application built with Node.js, Express, and EJS that provides detailed information about countries using the REST Countries API. This project demonstrates API integration, server-side rendering, and modern web development practices.

##  Features

-  **Search Any Country** - Find information by country name
- üá∫üá≥ **Comprehensive Data** - View capital, population, currency, languages, timezones, and more
-  **Quick Search Buttons** - One-click access to popular countries
-  **Responsive Design** - Works on all device sizes
-  **Real-time API Data** - Always current information
-  **Error Handling** - User-friendly error messages for invalid searches
-  **Random Country** - Discover new countries randomly

##  Technologies Used

- **Backend:** Node.js, Express.js
- **Frontend:** EJS Templates, HTML5, CSS3, JavaScript (ES6+)
- **HTTP Client:** Axios
- **API:** [REST Countries API](https://restcountries.com)
- **Package Manager:** npm
- **Version Control:** Git, GitHub
