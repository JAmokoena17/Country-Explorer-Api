# Country Explorer API

A full-stack web application that provides detailed information about countries using the REST Countries API.

## Features

- Search any country by name
- View detailed country information (capital, population, currency, languages, timezones, etc.)
- Quick search buttons for popular countries
- Responsive design

## Technologies Used

- Node.js
- Express.js
- EJS
- Axios
- REST Countries API

## Installation

1. Clone the repository: `git clone https://github.com/JAmokoena17/Country-Explorer-Api.git`
2. Install dependencies: `npm install`
3. Run the server: `node index.js`
4. Open `http://localhost:3000` in your browser.

## Usage

1. Enter a country name in the search box.
2. Click "Search" to view the country's information.
3. Use the quick buttons to search for popular countries.
